# OpenPGxLab

**OpenPGxLab** is an open learning and development hub for **pharmacogenomics (PGx)**, covering biological foundations, sequencing technologies, bioinformatics, HLA, annotation resources, and clinical interpretation.

https://aardes.github.io/OpenPGxLab/

## Overview

Pharmacogenomics connects genetics, pharmacology, bioinformatics, and clinical decision-making. OpenPGxLab is being built as a practical and community-oriented resource to help bridge these areas in a structured way.

The goal is not only to explain PGx concepts, but also to explore how PGx is tested, analyzed, interpreted, and implemented in real-world settings.

## What This Project Covers

- PGx foundations
- core PGx genes and mechanisms
- HLA and immune-mediated PGx
- genotype-to-phenotype interpretation
- PGx testing technologies
- bioinformatics pipelines and difficult loci
- annotation databases and decision support
- clinical reporting and implementation
- population-specific PGx challenges
- advanced and future PGx topics
